# xy-image

---

> 基于`React Hooks` + `typescript`的基础组件

## 安装

```sh
# npm
npm install --save xy-image

# yarn
yarn add xy-image
```

## 使用

```ts
import React from "react";
import ReactDOM from "react-dom";
import XyImage from "xy-image";
ReactDOM.render(<XyImage />, container);
```

## 开发

```sh
yarn run start
```

## 例子

http://localhost:6006

## 测试

```
yarn run test
```
